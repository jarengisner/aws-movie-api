Movie Findr API

The Movie Findr API is a REST API that provides information about movies, which are stored on a database. The API can provide details about individual movies, and the actors in those films.

GET

/movies/

returns a list of all movies within the database in JSON format

/movies/:title/

returns a list of all movies within the database, filtered by title in JSON format

/movies/:actors/

returns a list of all movies within the database, filtered by actors, in JSON format

/movies/:directors/

returns a list of all movies within the database, filtered by directors, in JSON format

/movies/:genre/

returns a list of all movies within the database, filtered by genre, in JSON format

/users/favorites/

returns a list of all movies within the current users favorites, in JSON format

/directors/

returns a list of all directors in the database, in JSON format

/actors/

returns a list of all actors in the database, in JSON format

/actors/:name/

returns a list of all actors within the database, filtered by name, in JSON format

/directors/:name/

returns a list of all directors in the database, filtered by name, in JSON format

POST

/users/register/

Returns a confirmation of registration, along with an HTTP status of 200.

/users/favorites/

Returns a confirmation that the movie was added to your favorites, with an HTTP status of 200

PUT

/users/:username

Returns confirmation of updated username, along with HTTP status code of 200

DELETE

/users/favorites/

Returns confirmation of a favorite delete, and returns JSON of updated list of favorites

/users/:username/

Returns confirmation that account was deleted, and HTTP status code of 200
